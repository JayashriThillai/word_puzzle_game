# Word Puzzle Game

An interactive word-scramble game developed using Python and Pygame.

## Features
- Random word scrambling
- Timer-based gameplay
- Score tracking system
- Hint functionality
- Background music and interactive UI

## Technologies Used
- Python
- Pygame
